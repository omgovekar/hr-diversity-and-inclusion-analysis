# HR Diversity & Inclusion Analysis (Power BI)

## Project Overview
This project analyzes workforce diversity, job levels, employee performance, promotions, hiring, and retention using HR data. The goal is to help HR teams identify diversity gaps, promotion trends, and retention risks.

## Key Objectives
- Analyze gender diversity across job levels
- Evaluate performance ratings by department and experience
- Track promotions and hiring trends
- Study employee retention and tenure patterns

## Dashboards Included
- Workforce Diversity Overview
- Job Level & Performance Analysis
- Promotion & Hiring Insights
- Retention & Employee Tenure

## Tools Used
- Power BI
- Excel

## Note
This is a simulated HR dataset used for learning and portfolio purposes.
